# 🚀 Featured Projects

## 📦 CargoGuard — Customs X-ray Intelligence Platform
**Role:** AI/ML & Backend Developer  
🔗 **Repo:** [CargoGuard](https://github.com/ShubhamDudhbhate-VJTI/Pixel.git)

### The Problem
Manual customs screening is slow and prone to human error, leading to missed prohibited items or manifest discrepancies.

### The Solution
An end-to-end AI platform that synchronizes **YOLOv8** object detection with **Zero-Shot Learning** to automatically flag undeclared cargo.

### 🛡️ Key Edge: Audit & Accountability
- **Visual Evidence:** Uses **Grad-CAM** heatmaps to explain AI detections, ensuring officers see *why* an item was flagged.
- **Decision Provenance:** A robust **Audit Trail** system that logs AI confidence scores, manifest comparison logs, and decision history for full transparency.
- **Multimodal Logic:** Directly compares extracted PDF manifest data with visual scan analysis to find mismatches instantly.

**Tech Stack:** FastAPI • React • YOLOv8 • PyTorch • OpenCV • LangChain
